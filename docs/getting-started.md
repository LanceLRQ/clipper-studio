# 简介

ClipperStudio 是一个跑在你自己电脑上的桌面软件，专门用来处理直播录播的切片工作。

它的核心思路很简单：**你把录播文件扔进来，它帮你找高能片段、切视频、出字幕、压弹幕，最后给你一个能直接投稿的成品。**

整个过程不需要联网（除了 AI 语音识别和插件这类可选功能），不会上传你的文件，也没有注册登录这类流程。装好就能用。

## 它能帮你做什么？

| 功能 | 一句话说明 |
|------|-----------|
| 工作区管理 | 把录播目录导进来，自动识别主播和场次 |
| 音量热度条 | 一眼看出直播里哪个时间段最热闹 |
| 视频切片 | 选好起止时间，一键切出来 |
| AI 字幕 | 自动识别语音生成字幕，支持编辑和搜索 |
| 弹幕压制 | 弹幕直接烧进视频，投稿就能用 |
| 插件扩展 | 想换引擎、加功能，装个插件就行 |

## 和其他工具有什么不同？

你可能用过一些切片工具，但大多会遇到这些问题：

- 参数太多，要自己研究 FFmpeg 怎么用
- 想找精彩片段只能手动拖进度条
- 每次操作都要开好几个软件来回切换
- 弹幕压制要自己装命令行工具

ClipperStudio 想做的事就是把这些麻烦都省掉。默认情况下你不需要调任何参数，点按钮就能出结果。如果你是进阶用户，高级选项也藏在折叠面板里，想调的时候随时能找到。
