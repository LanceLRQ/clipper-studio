# 视频工作台

视频工作台是 ClipperStudio 的核心页面，你在这里看视频、找片段、做切片。

## 打开工作台

在视频列表里点击任意一个视频，就会进入工作台页面。页面主要分两块：

- **上半部分** — 视频播放器
- **下半部分** — 取决于当前模式，可能是切片面板或字幕编辑器

## 播放视频

点击播放按钮就能看视频，这个没什么好说的。不过有几个小功能值得了解一下：

**键盘快捷键：**
- 空格 — 播放 / 暂停
- 左右方向键 — 快退 / 快进
- `I` 键 — 标记切片起点
- `O` 键 — 标记切片终点

**播放器支持的功能：**
- 多音轨切换（如果视频里有多个音轨的话）
- 倍速播放
- 字幕轨切换
- 弹幕叠加预览

## 音量热度条

这是 ClipperStudio 最实用的功能之一。在播放器下方你会看到一条彩色的时间轴：

```
灰色 ████████████████████████ 红色
低音量                          高音量
```

颜色越红，说明那个时间段的音量越大。直播里音量突然飙高的地方，往往就是高能时刻——主播在激动、观众在刷弹幕、有人在整活儿。

**怎么用：**
1. 扫一眼热度条，找红色最集中的区域
2. 点一下那个位置，播放器自动跳过去
3. 看看是不是你要找的片段
4. 如果是，直接标记起止时间做切片

比起手动拖进度条一帧帧找，这个效率高太多了。

::: tip
热度条的数据在导入视频时会自动生成，不需要你做什么。如果你导入了弹幕文件，热度条还会叠加弹幕密度信息，定位更精准。
:::

## 切片

找到想要的片段后，接下来就是切片了。

### 选择切片范围

有五种方式可以设置起止时间：

1. **快捷键** — 播放的时候按 `I` 设起点，按 `O` 设终点，最直观
2. **时间轴拖选** — 在时间轴上直接拖出一个选区
3. **手动输入** — 在切片面板里精确输入时间码，比如 `01:23:45`
4. **拖拽边界** — 已经设好选区后，拖动左右边界微调
5. **字幕辅助** — 点字幕里的某一行，自动选中那一段的时间范围

用哪种都行，顺手就好。

### 开始切片

设好范围后，点「开始切片」按钮就行了。

如果你想调参数，展开面板能看到一些选项：
- **编码预设** — 「快速」「标准」「高质量」三个档，一般选「标准」就够了
- **输出格式** — 默认 MP4，也可以选其他格式
- **是否包含弹幕** — 如果你导入了弹幕，可以选择是否压进视频里

不过说实话，大部分时候直接用默认值切就行，不需要动这些。

### 查看切片进度

切片任务提交后，你不需要干等着。页面下方有个任务面板，会显示当前任务的进度。你也可以继续浏览其他视频，任务会在后台排队处理。

切片完成后，输出文件会自动保存，在任务面板里可以点击打开文件所在目录。

### 失败 / 完成的任务也能重试

任务面板里失败的任务可以右键「重试」，已完成的任务也支持重新跑一次（比如你想换个参数再切一遍）。批量选择多个任务时支持一键批量重试，弹幕压制等参数会原样保留，不需要重新勾选。

## 浏览模式与编辑模式

工作台有两种模式，通过顶部的标签页切换：

**浏览模式（默认）** — 用来找片段和做切片，界面是播放器 + 热度条 + 切片面板。

**编辑模式** — 用来编辑字幕，界面变成播放器 + 字幕编辑器。详细操作看 [字幕系统](/subtitle)。

两个模式之间切换不会丢失你当前的进度，放心来回切。
