# 常见问题

这里收集了一些大家经常问到的问题。如果你的问题不在这里，可以去 [GitHub Issues](https://github.com/LanceLRQ/clipper-studio/issues) 提问。

## 安装相关

### macOS 提示「无法验证开发者」怎么办？

这是 macOS 的安全机制。打开「系统设置 → 隐私与安全性」，往下翻找到被拦截的应用，点「仍要打开」就行。

或者你也可以右键点击应用，选「打开」，在弹出的对话框里再点一次「打开」。

### Windows 杀毒软件报毒怎么办？

这是误报。ClipperStudio 是开源软件，你可以自己审查源代码。如果杀毒软件拦截了，把它加入白名单就好。

### 安装完打不开 / 闪退

检查几件事：
1. 你的操作系统版本是不是太旧了？ClipperStudio 需要 Windows 10+、macOS 12+ 或比较新的 Linux 发行版
2. 如果是 Windows，试试以管理员身份运行
3. 去看看日志文件有没有报错信息（日志位置见 [设置与依赖管理](/settings)）

## 功能相关

### FFmpeg 装不上怎么办？

依赖管理器里的自动下载偶尔会受网络影响。如果下载失败，你可以：

1. 自己去 [FFmpeg 官网](https://ffmpeg.org/download.html) 下载对应版本
2. 在依赖管理里选「手动指定路径」，指向你下载的 ffmpeg 文件

### 热度条没有颜色 / 全是灰色

这说明音量数据还没提取完。导入视频后，ClipperStudio 会在后台计算音量数据，比较长的视频需要等一会儿。等它算完就好了。

### 切片出来的视频画质不好

默认用的是「标准」预设，追求速度而不是极致画质。如果你对画质有要求：

1. 在切片面板里展开高级选项
2. 把编码预设切换到「高质量」
3. 切片速度会慢一些，但画质会好很多

### ASR 识别速度很慢

本地 ASR 的速度取决于你电脑的配置。如果太慢：

- 确认电脑有足够的内存（推荐 8G 以上）
- 关掉其他占资源的程序
- 或者考虑用远程 ASR 服务（需要联网）

### 弹幕和视频不同步

常见原因：

1. 弹幕文件和视频不是同一场直播的——确认一下文件是否对应
2. 录播有分段，弹幕文件只对应其中一段——检查弹幕文件的时间范围

如果以上都没问题，可以在弹幕设置里手动调整时间偏移。

## 数据与文件

### 我的原始视频文件会被修改吗？

不会。ClipperStudio 只读取你的原始文件，切片产出物会保存到单独的输出目录里。你的原始文件永远不会被动。

### 数据库文件在哪？

ClipperStudio 用 SQLite 存储数据，数据库文件在用户数据目录下（具体路径见 [设置与依赖管理](/settings)）。它很小，一般不会超过几十 MB。

### 可以把数据迁移到另一台电脑吗？

可以。复制用户数据目录到新电脑的相同位置就行。记得工作区里引用的视频文件路径也要对应上。

## 其他

### 支持什么视频格式？

常见格式都支持：MP4、FLV、MKV、MOV、AVI、TS 等。如果有 FLV 文件损坏的情况，ClipperStudio 会自动尝试修复。

### 支持什么语言？

界面目前支持中文和英文。ASR 识别支持的语言取决于你用的引擎。

### 收费吗？

ClipperStudio 完全免费开源。ASR 等外部服务可能有自己的收费策略，但这和 ClipperStudio 本身无关。

### 会自动更新吗？

会的。有新版本时 ClipperStudio 会提醒你，你可以选择立即更新或稍后再说。
