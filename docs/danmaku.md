# 弹幕系统

弹幕是直播录播切片里很重要的一部分。ClipperStudio 可以帮你预览弹幕效果，然后压进视频里。

## 导入弹幕

弹幕文件需要手动导入到工作区。目前支持 B 站的 XML 弹幕格式（就是录播姬导出的那种）。

导入方式：在工作区管理页面或者视频详情里，选择「导入弹幕」，选中弹幕文件就行。

如果你用的是录播姬，它的目录里通常会自动保存弹幕文件，导入工作区的时候 ClipperStudio 会尝试一起识别过来。

## 预览弹幕

在视频工作台里，播放器上方有一个弹幕叠加层。导入了弹幕之后，播放视频时会看到弹幕像在 B 站上一样飘过去。

这个预览功能的意义在于：**你在切片之前就能看到弹幕的密度和分布，方便判断这段内容适不适合做切片。**

如果弹幕太多看不清画面，可以调整弹幕的透明度和密度。

## 弹幕与热度条

如果你导入了弹幕文件，热度条的颜色会叠加弹幕密度信息。也就是说，热度条不只看音量，还会看弹幕密度。弹幕突然变多的地方，往往也是精彩时刻。

这意味着热度条的定位会更准确——高音量 + 高弹幕密度的时段，几乎肯定就是你要找的片段。

## 弹幕压制

「压制」就是把弹幕烧进视频画面里，变成视频的一部分。这样不管用什么播放器播放，弹幕都会显示出来。

### 操作步骤

1. 在视频工作台里选择好切片范围
2. 在切片面板里勾选「包含弹幕」
3. 点「开始切片」

ClipperStudio 会调用 DanmakuFactory 把弹幕转成 ASS 字幕格式，然后用 FFmpeg 把它压进视频里。

压制需要的时间比纯切片长一些，因为要渲染每一帧画面。一段 2 分钟的切片加弹幕压制，大概需要 30-60 秒。

### 压制 vs 外挂字幕

你可能会问：为什么要压进视频？不能直接挂字幕文件吗？

压进视频的好处是投稿到 B 站后弹幕直接可见，不需要再单独上传字幕文件。缺点是弹幕变成了画面的一部分，不能关掉。

如果你更倾向于可开关的弹幕效果，可以导出 ASS 字幕文件，然后单独上传到 B 站的字幕区。

## 弹幕密度分析

ClipperStudio 会自动计算弹幕的密度分布（每秒弹幕数），这个数据主要用于：

- 叠加到热度条上，辅助高能定位
- 在高级模式下查看弹幕密度曲线

未来版本还会加入基于弹幕内容的高能检测功能，敬请期待。

## 常见问题

**弹幕时间对不上视频怎么办？**

这可能是因为弹幕文件的时间戳和视频不同步。检查一下弹幕文件是不是和这个视频对应的同一场直播。

**弹幕太多 / 太少怎么办？**

在切片面板的弹幕设置里，可以调整弹幕的密度采样。弹幕太多时可以降采样（只显示一部分），太少时可以保留全部。

**支持其他平台的弹幕吗？**

目前内置支持 B 站 XML 格式。其他平台的弹幕可以通过插件来扩展。
