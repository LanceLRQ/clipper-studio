# 工作区管理

工作区是 ClipperStudio 里最基础的概念，理解了这个，后面的操作就都好说。

## 什么是工作区？

说白了，工作区就是**一个文件夹**。这个文件夹里放着你的录播文件，ClipperStudio 通过它来管理你的视频资源。

比如你之前用录播姬录直播，录播姬会把视频文件按一定结构存放在某个目录下。你把这个目录加进来当工作区，ClipperStudio 就能自动识别里面的主播、场次和视频文件。

## 创建工作区

第一次打开软件时，会有一个三步引导带你走完初始化：

1. **创建工作区** — 选一个文件夹，告诉 ClipperStudio 你的录播都放在哪
2. **安装依赖** — 检查 FFmpeg 等外部工具是否就绪（详见 [设置与依赖管理](/settings)）
3. **配置语音识别** — 可选，想自动出字幕就配一下（详见 [字幕系统](/subtitle)）

如果你只想先随便看看，每一步都能「跳过」，之后在设置里再补也来得及。

第一步里有两种创建工作区的方式：

1. **导入已有录播目录** — 如果你有现成的录播文件夹，选这个。选好后，ClipperStudio 会按你指定的适配器去识别里面的内容。

2. **创建空的工作区** — 选一个空文件夹，给它起个名字。之后你可以手动往里面放视频文件。

### 选择适配器

创建工作区时要选一个「适配器」，告诉 ClipperStudio 用什么规则来理解这个目录：

| 适配器 | 适用场景 |
|--------|---------|
| **录播姬（BililiveRecorder）** | 用录播姬录的目录，会自动识别主播、场次、弹幕文件 |
| **通用目录** | 任意视频目录，按文件结构平铺导入，不做主播/场次推断 |

选错也不要紧——进入工作区设置可以随时切换适配器，下次扫描就会按新规则重新解析。

导入完成后，页面会自动跳到视频列表。

## 切换工作区

界面顶部有一个工作区选择器，长这样：

```
ClipperStudio    [ 说说Crystal录播 ▼ ]
```

点开下拉菜单，可以：
- 切换到别的工作区
- 选「全部工作区」同时看所有视频
- 点「添加工作区」再导入一个新的

切换工作区后，视频列表和任务列表会自动过滤，只显示当前工作区的内容。

## 工作区里有什么？

一个工作区下面，ClipperStudio 会帮你整理出这样的结构：

- **主播** — 自动从录播目录中识别出来（如果目录结构包含主播信息的话）
- **场次** — 一场直播算一个场次，同一场次下的视频文件会归在一起
- **视频文件** — 实际的视频文件（FLV、MP4 等）

你在视频列表里看到的就是按这个层级组织好的内容。如果你只是想看所有视频，不关心分组，直接在列表里浏览就行。

## 添加新的录播

已经创建了工作区之后，还想往里加新的录播怎么办？两种方式：

1. **直接把文件放进工作区文件夹** — 在工作区设置里开启「自动扫描」后，新文件会被自动发现并导入
2. **手动触发重新扫描** — 在工作区管理页面点一下「重新扫描」，扫描进度会实时显示，过程中也可以随时取消

::: tip
ClipperStudio 支持多种录播工具的目录格式。目前内置了录播姬（BililiveRecorder）和通用目录两种适配器，其他来源可以通过插件扩展。
:::

### 路径冲突保护

如果你新加的工作区目录和已有工作区有重叠，ClipperStudio 会提示你存在交叉，避免同一份文件被两个工作区重复管理导致状态错乱。

### 孤立视频提示

工作区里如果有原始路径已经不存在的视频（比如你把文件移到了别处），它们会被标记为「孤立」，方便你清理或修复。

## 管理工作区

在设置页面可以管理工作区：

- **重命名** — 改个你容易认的名字
- **修改路径** — 如果你把文件夹移了位置，更新一下路径就好
- **更换适配器** — 切换录播姬 / 通用目录，下次扫描会按新规则解析
- **切片输出目录** — 留空时切片会默认输出到源视频旁边的 `clips/` 子文件夹，也可以指定一个统一目录
- **删除工作区** — 只是删除 ClipperStudio 里的记录，不会删你硬盘上的文件（如果只剩一个工作区，会提示你不要删掉最后一个）

放心操作，ClipperStudio 不会动你的原始文件。
