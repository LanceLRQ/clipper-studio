# 设置与依赖管理

ClipperStudio 的设置页面用来管理应用本身的行为和外部依赖。

## 依赖管理器

这是设置里最重要的一个功能。ClipperStudio 做切片、压弹幕这些事情，背后需要一些外部工具（主要是 FFmpeg 和 DanmakuFactory）。依赖管理器就是帮你管好这些工具的。

### 怎么用

打开设置页面，你会看到一个依赖状态列表：

| 依赖 | 状态 | 说明 |
|------|------|------|
| FFmpeg | 已安装 / 未安装 | 切片、弹幕压制、音量分析 |
| DanmakuFactory | 已安装 / 未安装 | 弹幕格式转换 |

如果某个依赖显示「未安装」，点旁边的「安装」按钮就行。ClipperStudio 会自动下载对应平台的版本，不需要你自己去找安装包。

::: tip
FFmpeg 是核心依赖，建议装好软件后第一件事就把它装上。没有 FFmpeg 的话切片功能用不了。
:::

### 手动指定路径

如果你电脑上已经装过 FFmpeg 或者 DanmakuFactory，不需要重复安装。在依赖管理里选择「手动指定路径」，然后选到你已安装的文件就行。

ClipperStudio 启动时会自动扫描系统的 PATH 环境变量，如果发现已经装好了，会直接识别到。

### 重新检测

如果装了新的依赖但 ClipperStudio 没检测到，点「重新检测」按钮就行。

## 应用设置

### 通用设置

- **语言** — 界面语言（目前支持中文和英文）
- **主题** — 深色 / 浅色 / 跟随系统
- **启动时行为** — 是否最小化到托盘、是否记住上次的工作区

### 切片设置

- **默认输出目录** — 切片文件保存到哪里
- **默认编码预设** — 快速 / 标准 / 高质量
- **文件名模板** — 切片文件怎么命名，支持变量替换

### 播放器设置

- **弹幕透明度** — 弹幕叠加层的透明度
- **弹幕密度** — 降采样级别
- **自动播放** — 打开视频时是否自动播放

## 配置文件在哪？

ClipperStudio 的配置文件存在用户数据目录下：

- **Windows** — `%APPDATA%/clipper-studio/`
- **macOS** — `~/Library/Application Support/clipper-studio/`
- **Linux** — `~/.config/clipper-studio/`

一般情况下你不需要手动编辑这些文件，通过设置页面操作就够了。但如果你想把配置备份一下或者迁移到另一台电脑，直接复制这个目录就行。
