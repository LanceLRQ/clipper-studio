# 安装与启动

## 下载安装

去 [GitHub Releases](https://github.com/LanceLRQ/clipper-studio/releases) 页面下载对应系统的安装包。

**Windows** — 下载 `.msi` 或 `.exe` 安装包，双击运行，一路下一步就行。

**macOS** — 下载 `.dmg` 文件，打开后把应用拖进「应用程序」文件夹。首次打开时如果提示「无法验证开发者」，去「系统设置 → 隐私与安全性」里点一下「仍要打开」就好。

**Linux** — 下载 `.AppImage` 文件，加上执行权限后直接运行：
```bash
chmod +x ClipperStudio-*.AppImage
./ClipperStudio-*.AppImage
```

## 首次启动

第一次打开 ClipperStudio，会看到一个欢迎引导页面。它会让你做一件事：**导入或创建一个工作区**。

工作区就是一个文件夹，里面放你的录播文件。如果你之前用录播姬录过直播，直接选那个文件夹就行，ClipperStudio 会自动识别里面的内容。

具体的导入操作可以看 [工作区管理](/workspace) 那一章。

## 需要额外安装什么吗？

大部分功能开箱即用，但有两个东西是按需安装的：

| 依赖 | 干什么用 | 不装会怎样 |
|------|---------|-----------|
| **FFmpeg** | 切片、弹幕压制、音量分析都靠它 | 无法切片 |
| **DanmakuFactory** | 把弹幕文件转成 ASS 字幕格式 | 弹幕功能用不了 |

ClipperStudio 会在启动时自动检测这些工具有没有装好。如果没装，它会在设置页面提示你，大部分情况下点一下就能自动下载安装。详细的说明看 [设置与依赖管理](/settings)。

::: tip
FFmpeg 是必装的，没有它切片功能跑不起来。建议在第一次使用前就去设置页面把它装好。
:::

## 界面概览

装好之后，你会看到这样的界面布局：

- **左侧边栏** — 文档目录导航，快速跳转
- **顶部** — 工作区切换器、搜索框
- **中间主区域** — 视频列表 / 播放器 / 任务面板
- **右侧面板** — 字幕编辑、切片参数等

不用死记这些，用着用着就熟了。
