## 这软件是给谁用的？

如果你经常做的事情是——打开一个三五个小时的直播录像，来回拖进度条找有意思的片段，切出来压上弹幕然后投到 B 站——那这个工具就是为你做的。

ClipperStudio 把这套流程搬到本地桌面应用里，让你不用再开一堆窗口、装一堆命令行工具。能离线用，不强制注册账号，不把你的文件往云端传。

## 能做什么？

简单来说，ClipperStudio 帮你完成这几件事：

1. **导入录播** — 把已有的录播文件夹导进来，自动识别里面的内容
2. **快速定位** — 用音量热度条找到直播里气氛最嗨的片段
3. **切片** — 选好时间段，一键切出想要的片段
4. **生成字幕** — 用 AI 语音识别自动出字幕，也能手动编辑
5. **弹幕压制** — 把弹幕烧进视频里，直接拿去投稿
6. **批量处理** — 多个视频排队处理，不用一个个盯着
